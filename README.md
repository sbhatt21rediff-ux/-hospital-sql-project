# -hospital-sql-project
PostgreSQL Hospital Database Project with SQL Queries

This project demonstrates SQL queries written in PostgreSQL
using a Hospital dataset.

## Files
- schema.sql – Table creation and data loading
- queries.sql – SQL queries for analysis
- Hospital_Data.csv – Dataset used in the project

## CSV file is stored inside the `data/` folder.


## Database
PostgreSQL

## Description
This project was created as part of a SQL practice assignment
to demonstrate aggregation, grouping, and date functions.
